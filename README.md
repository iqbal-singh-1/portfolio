# Iqbal Singh's Portfolio

Welcome to my personal portfolio! This project is a visually stunning, fully responsive web application built to showcase my journey as a software developer, the projects I've built, and my real-time coding statistics.

## ✨ Features

- **Dynamic GitHub & LeetCode Stats:** Fetches and displays real-time statistics directly from GitHub and LeetCode APIs, so my coding activity is always up-to-date.
- **Glassmorphic UI Design:** A beautiful, modern design featuring glassmorphism (translucency, blur, glowing borders), combined with a dark theme and interactive constellation background.
- **Responsive Layout:** fully optimized for desktops, tablets, and mobile devices to ensure a seamless experience everywhere.
- **Open Source Showcase:** Aesthetic, custom-built repository cards to display my code-focused projects and open source work.
- **Achievements & Certifications Section:** Visually appealing timelines and achievement boards highlighting key career milestones.

## 🛠️ Technology Stack

- **Framework:** [React 19](https://react.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** Vanilla CSS (CSS3) with advanced transitions and backdrop-filters
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites

You will need to have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/iqbal-singh-1/portfolio.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd portfolio/my-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to the local URL provided by Vite (usually `http://localhost:5173/`).

## 🖼️ Structure

- `src/App.tsx`: The main root component that orchestrates navigation and global layout.
- `src/index.css`: Contains the entire global design system, the glassmorphic tokens, and responsive utility media queries.
- `src/components/`: Modular, self-contained sections of the portfolio:
  - `About.tsx`: My educational background and journey.
  - `Stats.tsx`: Live-fetching coding profile widgets.
  - `Projects.tsx`: Showcase of Open Source Work.
  - `Certifications.tsx`: Earned achievements and credentials.
- `src/assets/`: Holds static resources such as the hero images and my CV.
